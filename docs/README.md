## 🎯 ゲーム仕様

### コアゲームルール
1. **移動**: タッチ・クリックで機体移動
2. **射撃**: 自動射撃システム
3. **敵撃破**: スコア獲得
4. **パワーアップ**: 
   - 弾で撃破 → 正効果（攻撃力アップ等）
   - 直接接触 → 負効果（攻撃力ダウン）
5. **ヘルス**: HPが0でゲームオーバー

### 敵の種類（Phase 3で実装予定）
- **Basic**: 直線移動
- **Fast**: ジグザグ移動
- **Tank**: 高耐久・低速
- **Boss**: 複雑攻撃パターン
