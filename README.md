# Mobile Last War v2

シンプルで直感的なモバイル向けシューティングゲーム

## 🎮 ゲーム概要

Mobile Last War v2は、ブラウザで遊べるシンプルなシューティングゲームです。左右移動のみのシンプル操作で、時間と共に難易度が上がる敵を撃破しましょう！

### 🎯 ゲームの特徴

- **シンプル操作**: 画面タッチで左右移動のみ
- **自動連射**: 手動射撃不要の快適プレイ
- **時間ベース難易度**: 30秒毎に段階的に難しくなる
- **ワンタップゲームオーバー**: 敵がプレイヤーエリアに到達で即終了
- **パワーアップシステム**: 弾で撃破vs直接接触の戦略性

### 🕹️ 操作方法

- **移動**: 画面タッチ/マウス移動で左右移動のみ
- **射撃**: 自動連射（操作不要）
- **ポーズ**: 右下の⏸ボタン

### 🎯 ゲームルール

1. **基本ルール**
   - プレイヤーは画面下部に固定
   - 敵を撃破してスコアを獲得
   - 敵がプレイヤーエリア（画面下部80%）に到達で即ゲームオーバー

2. **パワーアップシステム**
   - 敵撃破時に一定確率でアイテム出現
   - **弾で撃破**: 攻撃力アップ等の良い効果
   - **直接触る**: 攻撃力ダウンの悪い効果

3. **難易度システム**
   - **0〜29秒**: 1秒に1体の敵
   - **30〜59秒**: 0.8秒に1体
   - **60〜89秒**: 0.6秒に1体
   - **90〜119秒**: 0.4秒に1体
   - **120秒以降**: 0.2秒に1体

## 🎮 ゲームプレイのコツ

1. **パワーアップは弾で撃破**
   - 直接触ると攻撃力が下がります
   - 弾で撃破すると攻撃力等がアップ

2. **敵の到達を防ぐ**
   - 敵がプレイヤーエリアに到達すると即ゲームオーバー
   - 上手く左右移動して敵を撃破しましょう

3. **時間経過に注意**
   - 30秒毎に敵の出現頻度がアップ
   - 2分経過すると0.2秒に1体の高難易度に
